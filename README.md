# Bank Management System

## Project Overview

The Bank Management System is a menu-driven C++ application developed using Object-Oriented Programming (OOP) concepts. The system manages different types of bank accounts and allows users to perform banking operations such as deposit, withdrawal, balance inquiry, and interest calculation.

This project demonstrates the implementation of:
- Classes and Objects
- Encapsulation
- Inheritance
- Polymorphism

---

## Features

- Create and manage different account types:
  - Savings Account
  - Checking Account
  - Fixed Deposit Account

- Deposit money into accounts
- Withdraw money from accounts
- Display account details
- Calculate interest for Savings and Fixed Deposit accounts
- Overdraft facility for Checking accounts
- Menu-driven user interface
- Runtime Polymorphism using base class pointers

---

## OOP Concepts Used

### 1. Class and Object
Classes are used to represent different account types, and objects are created to manage account information.

### 2. Encapsulation
Account balance is declared private and can only be accessed through getter and setter methods.

### 3. Inheritance
Derived classes inherit common properties and methods from the BankAccount base class.
